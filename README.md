# tennis-book 网球相关书籍

本仓库收集**以简体中文（中国大陆版）为主**的网球图书书目，便于检索、引用与后续扩展。

## 数据文件

| 文件 | 说明 |
|------|------|
| [`data/books-zh-cn.json`](data/books-zh-cn.json) | 结构化书目（书名、作者、译者、出版社、ISBN、豆瓣条目链接等） |

## 范围说明

- 优先收录有正式简体中文版出版的图书；信息主要对照 [豆瓣读书](https://book.douban.com/) 等公开书目页整理。
- 若同一原著存在多个简体版本（例如阿加西《网》与《阿加西自传》），在条目中用 `note_zh` 标明关系。
- **未纳入**：目前仅能确认以英文版或繁体版流通、尚无可靠简体版权信息的图书（例如部分国外新书）未写入列表，避免误标为「中文版」。

## 授权

书目为元数据汇总；图书正文版权归各出版社及作者所有。
