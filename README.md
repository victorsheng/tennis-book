# tennis-book

这里收集的是一组网球主题电子书与训练资料，适合从技术学习、专项训练到阅读名将故事时按主题查找。

## 这套书里有什么

- 技术与战术：击球动作、步法、截击、双打抢打、实战策略
- 训练与心理：压力训练、专项能力、训练组织、比赛状态
- 身体与科学：拉伸、解剖、疼痛缓解、损伤预防
- 名将传记：阿加西、纳达尔、德约科维奇、费德勒
- 规则与测评：CTN 等级测评与基础参考资料

## 阅读方式

仓库根目录保存 PDF 文件，[docs/index.html](/Users/victor/code-me/books/docs/index.html) 提供了一个双语书目页，可以按关键词和分类快速找到资料。

如果需要本地打开书目页，可以运行：

```bash
python3 -m http.server 8000 --directory docs
```

然后访问 [http://localhost:8000/](http://localhost:8000/)。
